```
$ git clone https://github.com/yahoo/navi.git
$ cd navi
$ cd packages/webservice
$ ./gradlew bootRun

** Ctrl-C to Exit **
```

***Congratulations you are all set, you can now Open <a href="http://localhost:8080"> http://localhost:8080</a>.***
